# Курсовой проект "Сервис перевода денег"

Для запуска склонируйте (или просто [скачайте](https://github.com/lamasurfer/jclo_transfer_docker/archive/refs/heads/main.zip)) репозиторий:
```bash
git clone https://github.com/lamasurfer/jclo_transfer_docker.git 
```

В корневом каталоге запустите:
```bash
docker-compose up -d
# время сборки - 2-3 минуты
```

Сервис будет доступен по адресу: http://localhost

Для завершения работы:
```
docker-compose down
```
